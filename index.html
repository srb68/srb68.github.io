AI PDF Study Assistant

code
March 2026

A RAG-powered study tool that lets you upload PDFs and have an actual conversation with them. Ask questions, get summaries, drill into specific sections — the AI always cites which part of the document it's pulling from.

The backend uses FAISS for vector similarity search over chunked document embeddings, with Google Gemini handling the generation side. I built a FastAPI server that manages document ingestion, embedding storage, and query routing. The React frontend has a clean chat interface where you can upload multiple PDFs, switch between documents, and see highlighted source citations inline with responses.

One thing I focused on was retrieval quality — chunking strategy matters a lot. I experimented with different chunk sizes and overlap settings and ended up with a sliding window approach that keeps context intact across section boundaries. Response relevance improved noticeably compared to naive fixed-size chunking.

Python FastAPI FAISS Google Gemini API React RAG Vector Embeddings

Battlesnake — Thunga 🐍

code
2026 · Hackathon Tournament

A competitive Battlesnake AI built for a hackathon tournament. Thunga evaluates every possible move each turn using a multi-factor scoring system with a hard 200ms time budget — it doesn't just avoid walls, it profiles opponents and adapts its strategy in real time.

  • A* Pathfinding — Optimal paths to food using Manhattan distance heuristic, accounting for occupied cells and tail movement prediction.
  • Flood Fill — Calculates reachable space from any position to avoid dead ends and evaluate long-term survivability.
  • Voronoi Space Control — Multi-source BFS from all snake heads simultaneously to compute board territory ownership, prioritizing moves that maximize controlled space.
  • Opponent Profiler — Tracks enemy behavior across turns and classifies opponents into archetypes: vacuum, camper, aggressor, area_control, erratic. Each triggers specific countermeasures — blocking a vacuum from food, avoiding erratic snakes at close range, or tail-chasing against an aggressor.
  • Time-Budgeted Engine — Evaluates in priority order and aborts deep evaluation if elapsed time exceeds 200ms, always returning a valid response within Battlesnake's timeout window.
Python FastAPI A* BFS / Flood Fill Voronoi BFS Docker
